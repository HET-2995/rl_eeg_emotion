# RL Final Project - EEG Emotion Recognition

This repository contains the implementation of a Reinforcement Learning approach for EEG-based emotion recognition using the DEAP dataset. 

## Project Structure

* `baseline.ipynb`: Baseline model implementation for emotion recognition.
* `inovation.ipynb`: Innovative approaches and improved models.
* `final_rl (3).pdf`: Project report and analysis.
* `deap-dataset/`: Contains the DEAP dataset files used for training and evaluation.
* `outputs_baseline/`: Generated outputs from the baseline models.
* `outputs_innovation/`: Generated outputs from the innovative models.

## Dataset

The DEAP (Database for Emotion Analysis using Physiological Signals) dataset is used in this project. Ensure the dataset is correctly placed in the `deap-dataset` directory before running the notebooks.

## Running the Code

1. Install the required dependencies (e.g., PyTorch, stable-baselines3, numpy, pandas).
2. Run `baseline.ipynb` to evaluate the baseline performance.
3. Run `inovation.ipynb` to see the improvements using the proposed RL techniques.
